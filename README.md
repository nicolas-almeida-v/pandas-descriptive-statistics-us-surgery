# Descriptive Statistics with Pandas 📊

## About this project
This is my second data analysis project. Using the cleaned US surgery dataset from my 
previous project, I explored and summarized the data using descriptive statistics 
techniques with Python and Pandas in Google Colab.

## What this notebook covers
- Examining the dataframe
- Counting and frequencies
- Grouped frequencies
- Measures of central tendency — arithmetic mean, geometric mean, median and mode
- Measures of dispersion — standard deviation, range and quantiles
- Normality assessment
